Vue框架学习

2023-05-05 前端学习 Vue 评论

vue官网

https://cn.vuejs.org/

Vue基础

创建vue项目

使用vue create 项目名进行创建

或者使用vue ui进行创建（常用）

基础语法

  1. 插值表达式
  2. v-bind、v-model、v-show、v-if、v-else-if、v-else、v-on 、v-html、v-for的使用
  3. axios请求，npm install axios

解决跨域问题

在vue.config.js文件中配置代理，所有api下的请求都转发到8080端口，并且去除掉/api

Router路由

创建时就添加vue-router的依赖，也可以npm install vue-router@^3.5.1

router大致可以分为三部分：VueRouter路由器(根据路由请求渲染视图)、router-link路由连接组件、router-view路由视图组件

使用：

在src目录下新建router目录，里面新建index.js文件，注册路由器和路由规则

在main.js中使用路由规则

在使用的位置加上

router的跳转除了标签式的跳转，还有编程式的跳转

如果请求资源不存在，进行统一跳转

嵌套路由

这样就可以实现嵌套路由

ElementUI

npm i element-ui -S 安装elementui

在main.js中导入ElementUI

ElementUI官网组件：https://element.eleme.cn/#/zh-CN/component/installation

Vuex

vuex是为vue.js应用开发的状态管理库，可以用于多个组件之间进行共享数据和集中管理组件状态。

调用mutation中的方法修改共享数据，需要使用this.$state.commit(‘setName’,’李四’),来调用，setName是方法名，李四是参数，但是mutation中只能是同步方法，异步方法需要写在actions中。

TypeScript

是微软推出的一款开源语言，涵盖js的全部，可以被编译为js，在编译时进行校验，可以使编码更规范。

ts VS js

  • ts是静态编程语言，js是动态类型编程语言
  • 静态类型在编译期做类型检查，动态类型语言在执行期进行检测
  • 对于js而已，其发现问题较晚，ts较早
  • 配合vscode可以提前发现代码错误

全局安装ts，npm install -g typeScript

在创建项目时就导入typeScript，ts需要编译,tsc xxx.ts

ts常用类型

string、number、boolean、void、interface、class

ts并不能直接运行，需要经过编译为js文件，会擦除掉类型转为js

环境搭建

一般都需要导入以下技术

  • ElementUI
  • axios
  • vuex
  • vue-router
  • typescript

这些都可以在vue-ui的地方直接导入
